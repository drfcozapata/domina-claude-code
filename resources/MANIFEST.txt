💡 CONSEJOS PROFESIONALES

• Usar favicon-32.svg como favicon principal (funciona en todos los navegadores modernos)
• Usar icons/domina-claude-code-icon.svg para avatares a 64-256px
• Usar logos/domina-claude-code-logo.svg para encabezados de sitio web (escalar para ajustar)
• Mantener el concepto de hexágono ascendente para cualquier variante personalizada
• Siempre mantener la paleta de colores establecida
• Los archivos SVG son editables - ver DESIGN-SYSTEM.md para personalización de color
• No se necesita conversión a PNG - SVG funciona en todas partes
